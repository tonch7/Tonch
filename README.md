# FinApp

FinApp é um sistema de gestão financeira simples, rápido e totalmente gratuito.  
Criado para quem precisa controlar receitas, despesas e contas de forma prática, sem depender de aplicativos pesados, anúncios ou assinaturas.

O projeto foi desenvolvido com foco em **simplicidade, leveza e controle total dos dados pelo usuário**.

---

## Sobre o Projeto

A ideia do FinApp surgiu da necessidade de ter uma ferramenta direta para organizar a vida financeira sem complicações.

Muitas plataformas existentes são complexas, cheias de anúncios ou exigem assinaturas.  
Este sistema foi criado para ser **leve, funcional e acessível para qualquer pessoa**.

O FinApp roda diretamente no navegador e pode ser hospedado em qualquer servidor que suporte PHP.

---

## Principais Recursos

- Dashboard financeiro
- Controle de receitas
- Controle de despesas
- Gestão de contas bancárias
- Relatórios financeiros
- Exportação de dados
- Interface simples e rápida
- Banco de dados leve utilizando SQLite

---

## Tecnologias Utilizadas

- PHP
- SQLite
- HTML5
- CSS3
- JavaScript

---

## Objetivos do Projeto

- Oferecer uma ferramenta financeira simples e eficiente
- Manter o sistema leve e rápido
- Evitar dependência de aplicativos externos
- Garantir total controle dos dados pelo usuário
- Disponibilizar o código para a comunidade
