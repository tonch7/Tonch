# 💰 FinApp

> Controle financeiro simples, direto e seu. Sem anúncios. Sem rastreio.
> Sem porcaria.

O **FinApp** é um aplicativo de gestão financeira pessoal e empresarial
desenvolvido com foco em simplicidade, autonomia e privacidade.

## 🚀 Filosofia do Projeto

-   ❌ Sem anúncios\

-   ❌ Sem coleta de dados\

-   ❌ Sem dependência de serviços externos

-   ✅ 100% seu\

-   ✅ Offline-first\

-   ✅ Leve e rápido\

-   ✅ Código aberto (MIT)

## 🧠 O que o FinApp faz

-   Controle de receitas e despesas\
-   Contas a pagar e receber\
-   Dashboard financeiro\
-   Relatórios\
-   Exportação PDF\
-   Sistema de login\
-   Banco local (SQLite)\
-   Vault seguro

## 🛠️ Tecnologias

-   PHP
-   SQLite
-   HTML / CSS / JS

## ⚙️ Como rodar

git clone https://github.com/seu-usuario/finapp.git

Coloque em: C:`\xampp`{=tex}`\htdocs`{=tex}`\finapp`{=tex}

Acesse: http://localhost/finapp/public
